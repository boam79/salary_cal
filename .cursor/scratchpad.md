# 금융 계산기 프로젝트

## 프로젝트 개요
10가지 금융 계산기와 종합뉴스를 제공하는 SPA 웹앱

## 최근 작업 내역

### ✅ 2025-11-03: 근로소득공제 및 로또 최적화
- **근로소득공제 2025년 기준 수정** (500만→600만원)
- **사람인 계산기 비교 분석** (표준 vs 실용)
- **로또 전체 데이터 수집** (1~1196회, 156KB)
- **Render 무료 플랜 최적화** (크론, 웜업, 타임아웃)

---

## 주요 기능

### 계산기 (10종)
1. 월급/연봉 - 2025년 세율 기준
2. 세금 - 상속세, 증여세
3. 부가세 - 공급가액↔총액, 환급
4. 부동산 - 중개수수료, 양도세, 보유세, DSR
5. 취등록세 - 부동산 취득/등록세
6. 대출 - 원리금균등/원금균등/만기일시
7. 적금/예금 - 단리/복리, 이자세
8. 퇴직금 - 입퇴사일 기반
9. 자동차 취득세 - 차종별 세율
10. 로또번호 - 빈도 기반 생성

### 추가 기능
- 📰 종합뉴스 RSS
- 📊 Canvas 그래프
- 📱 반응형 UI
