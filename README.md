## 🎯 프로젝트 개요

**금융 계산기 웹사이트**는 일상에서 필요한 다양한 금융 계산을 쉽고 빠르게 할 수 있도록 도와주는 웹 애플리케이션입니다.

### 핵심 특징

- ✅ **5가지 주요 계산기** (연봉, 세금, 부동산, 취등록세, 대출)
- ✅ **SPA 방식** 단일 페이지 애플리케이션으로 빠른 화면 전환
- ✅ **계산 과정 설명** 왜 그렇게 계산되었는지 상세 설명 제공
- ✅ **반응형 디자인** 모바일, 태블릿, 데스크톱 모두 지원
- ✅ **흑백 모던 UI** 깔끔하고 세련된 디자인
- ✅ **그래프 시각화** 대출 상환 스케줄을 시각적으로 표현
- ✅ **순수 JavaScript** 외부 라이브러리 없이 가볍고 빠름

## 🚀 주요 기능

### 1️⃣ 월급 및 연봉 계산기
- **입력**: 
  - 연봉 계산: 연봉 (만원 단위)
  - 월급 계산: 주간 근무시간, 시급 (2025년 최저시급 10,030원 기준)
- **출력**: 월/연 실수령액, 공제 내역
- **계산 항목**:
  - 소득세 (2025년 누진세율 적용)
  - 지방소득세 (소득세의 10%)
  - 4대보험 (국민연금, 건강보험, 장기요양보험, 고용보험)
  - 주휴수당 (주간급여의 12.5%)
- **특징**: 과세표준 계산 과정 상세 설명, 2025년 최저시급 자동 적용

### 2️⃣ 세금 계산기
#### 상속세 계산
- **입력**: 상속 재산가액, 공제 항목
- **출력**: 상속세액
- **특징**: 누진세율 구간별 계산

#### 증여세 계산
- **입력**: 증여 재산가액, 증여자 관계
- **출력**: 증여세액
- **특징**: 관계별 공제액 자동 적용

### 3️⃣ 부동산 계산기
#### 중개수수료 계산
- **입력**: 부동산 가격 (억원 단위)
- **출력**: 총 중개수수료, 매수자/매도자 부담액, 적용 수수료율

#### 양도소득세 계산
- **입력**: 취득가액, 양도가액, 보유기간
- **출력**: 양도소득세, 양도차익, 장기보유특별공제, 과세표준

#### 보유세 계산
- **입력**: 부동산 가치, 부동산 유형
- **출력**: 연간 보유세 총액, 재산세, 종합부동산세

#### DSR 계산
- **입력**: 연소득, 대출금액, 연이자율
- **출력**: DSR 비율, 월/연간 상환액, DSR 평가

### 4️⃣ 취등록세 계산기
- **입력**: 부동산 유형, 취득가액, 면적
- **출력**: 취득세, 등록면허세, 지방교육세
- **특징**: 주택/토지 구분, 면적별 세율 차등 적용

### 5️⃣ 주택 대출 및 금융대출 계산기 (드롭다운 메뉴)

#### 🏦 금융대출 계산기
- **입력**: 대출금액, 이자율, 대출기간, 상환방식
- **상환방식**:
  - 원리금균등상환 (매월 동일 금액)
  - 원금균등상환 (원금 균등 분할)
  - 만기일시상환 (이자만 납부, 만기 시 원금 상환)
- **출력**: 
  - 월 상환액
  - 총 이자
  - 상환 스케줄 테이블
  - **📊 그래프**: 원금(검은색)/이자(회색) 구분 시각화

#### 🏠 주택대출 계산기
- **입력**: 주택가격, 본인자금, 연소득, 이자율, 대출기간
- **계산**: 
  - **LTV (Loan To Value)**: 주택가격 대비 대출가능액
  - **DTI (Debt To Income)**: 소득 대비 부채상환비율
  - 월 상환액, 총 이자
- **출력**: 
  - 최대 대출가능액
  - 적정 대출액
  - 상환 스케줄 테이블
  - **📊 그래프**: 원금(검은색)/이자(회색) 구분 시각화
- **특징**: 주택담보대출 특화 조건 반영

### 대출 상환 공식

#### 원리금균등상환
```
월 상환액 = P × r × (1+r)^n / ((1+r)^n - 1)
P: 대출원금, r: 월 이자율, n: 총 상환개월 수
```

#### 원금균등상환
```
월 원금 상환액 = P / n
월 이자 = 잔여원금 × r
월 상환액 = 월 원금 + 월 이자
```

#### 만기일시상환
```
월 이자 = P × r
만기 상환액 = P + (P × r × n)
```

### 주택대출 비율 (2025년 기준)
- **LTV (주택담보인정비율)**: 일반적으로 40~70%
- **DTI (총부채상환비율)**: 일반적으로 40~60%

- **종합부동산세**: 6억 초과 시 추가 과세 (0.6%~1.6%)
