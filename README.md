## 🎯 프로젝트 개요

**금융 계산기 웹사이트**는 일상에서 필요한 다양한 금융 계산을 쉽고 빠르게 할 수 있도록 도와주는 웹 애플리케이션입니다.

### 핵심 특징

- ✅ **10가지 계산기** (연봉, 세금, 부가세, 부동산, 취등록세, 대출, 적금/예금, 퇴직금, 자동차 취득세, 로또)
- ✅ **🗞️ 최신 종합뉴스** RSS 피드 연동, 5분마다 자동 업데이트
- ✅ **ES6 모듈 시스템** 현대적인 코드 구조
- ✅ **SPA 네비게이션** 빠른 화면 전환
- ✅ **계산 과정 설명** 상세한 단계별 설명
- ✅ **반응형 3-Column 레이아웃** 광고 공간 포함
- ✅ **모던 UI/UX** 부드러운 애니메이션
- ✅ **그래프 시각화** Canvas API 기반
- ✅ **순수 JavaScript** 외부 라이브러리 없이 가볍고 빠름

## 🚀 주요 기능

### 1️⃣ 월급 및 연봉 계산기
- 연봉/월급 계산: 소득세, 지방소득세, 4대보험 공제
- 2025년 기준 누진세율 및 근로소득공제 적용
- 주휴수당 포함 월급 계산

### 2️⃣ 세금 계산기
- 상속세: 누진세율 구간별 계산
- 증여세: 관계별 공제액 자동 적용

### 3️⃣ 부가세 계산기
- 일반 계산: 공급가액 ↔ 총액
- 사업자 환급: 납부/환급 자동 산출

### 4️⃣ 부동산 계산기
- 중개수수료, 양도소득세, 보유세, DSR 계산
- 3가지 상환방식 지원

### 5️⃣ 취등록세 계산기
- 부동산 유형별 세율 적용
- 취득세 + 등록면허세 + 지방교육세

### 6️⃣ 대출 계산기
- 금융대출: 원리금균등, 원금균등, 만기일시상환
- 주택대출: LTV/DTI 자동 계산
- Canvas 그래프 시각화

### 7️⃣ 적금/예금 계산기
- 단리/복리 일시예치, 월 적립식
- 이자소득세 15.4% 반영

### 8️⃣ 퇴직금 계산기
- 입사일/퇴사일 기반 자동 계산
- 퇴직소득세 자동 계산

### 9️⃣ 자동차 취득세 계산기
- 차량 유형별 세율 (승용/영업용/이륜/하이브리드/전기)
- 취득세 + 지방교육세

### 🔟 로또번호 생성기
- 숫자 출현 빈도 기반 가중 샘플링
- 과거 당첨 조합 제외, 10세트 자동 생성
- 실제 로또 용지 형태 UI
- **주의**: 오락 목적, 당첨 보장하지 않음

### 🗞️ 종합뉴스 섹션
- RSS 피드 연동: 연합뉴스, KBS, MBC, SBS, JTBC 등
- 5분마다 자동 업데이트
- 정사각형 카드 UI, 5열 그리드

## 🛠 기술 스택

### Frontend
- **HTML5**, **CSS3**, **Vanilla JavaScript (ES6+)**
- **ES6 Modules**, **Singleton Pattern**, **Event Delegation**
- **Canvas API** - 그래프 렌더링 (외부 라이브러리 없음)

### 배포
- **Vercel** - 자동 배포
- **Render** - 로또 백엔드 서버

## 📊 계산 로직 및 세율 (2025년 11월 기준)

### 소득세 (누진세율)
| 과세표준 | 세율 | 누진공제 |
|---------|------|----------|
| 1,200만원 이하 | 6% | - |
| 1,200~4,600만원 | 15% | 108만원 |
| 4,600~8,800만원 | 24% | 522만원 |
| 8,800만원~1.5억 | 35% | 1,490만원 |
| 1.5억~3억 | 38% | 1,940만원 |
| 3억~5억 | 40% | 2,540만원 |
| 5억~10억 | 42% | 3,540만원 |
| 10억 초과 | 45% | 6,540만원 |

### 근로소득공제 (2025년)
| 연소득 | 공제율 |
|-------|--------|
| 600만원 이하 | 70% |
| 600~1,500만원 | 420만원 + 초과분 × 40% |
| 1,500~3,000만원 | 780만원 + 초과분 × 15% |
| 3,000~5,000만원 | 1,005만원 + 초과분 × 8% |
| 5,000~8,800만원 | 1,165만원 + 초과분 × 6% |
| 8,800만원 초과 | 1,393만원 + 초과분 × 2% |

### 4대보험
- **국민연금**: 4.5% (상한액: 553만원)
- **건강보험**: 3.545%
- **장기요양보험**: 건강보험료의 12.27%
- **고용보험**: 0.9%

### 최저시급
- **2025년**: 10,030원
